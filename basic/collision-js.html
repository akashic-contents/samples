collision-js によるヒット判定

collision-js は算術ライブラリを含むコリジョンライブラリです。 collision-js を利用すると矩形や円といったシェイプ同士が重なっているかどうかを判定できます。

青い物体をドラッグして移動することができます。緑の物体と重なると色が赤く変わります。 右上の 三 ボタンを押下すると交差判定するシェイプの組み合わせを選択することができます。

ダウンロード

動かし方

ダウンロードした zip ファイルを解凍して以下のコマンドを実行してください。

npm install
akashic-sandbox .

機能

collision-jsには以下の機能があります。

  • コリジョン
    • 以下のシェイプ間の交差判定。
      • 点
      • 円
      • 有向ボックス
      • 軸平行ボックス
      • 直線
      • 線分
  • 算術
    • ２次元と３次元のベクトル。
    • ２行２列と３行３列の行列。

詳しくはAPI リファレンスを参照ください。
